# Blogs

This folder contains blog posts written in Markdown. The files stored here are rendered via GitHub's Markdown API and published at https://yashgajera.io/blogs.

✨ Quick Details
- **Location:** `blogs/` — Markdown files here.
- **Rendering:** Rendered by GitHub's API and displayed on the site.
- **Filename:** `YYYY-MM-DD-title.md` for easy sorting.
- **Front matter:** Include `title`, `date`, and `tags` in YAML at the top if useful.

Questions or ideas? Open an issue or reach out via the site.
